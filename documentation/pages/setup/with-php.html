PHP Examples, API endpoint URL with parameters

ExcelWebDatabases uses an API with GET requests which returns JSON. The API endpoint is:

The parameters are:

Key is your API key. Mode is read/write, defined by r or w. Document is the name of the document you would like to access. isMulti is a boolean, on the basis of whether you are accessing an individual cell or multiple cells in one request. Cell is an Excel cell, ex. A4, B1, etc. Data is only provided if it is a write request, and is JSON.

Read requests

When making a read request: If you are reading a single cell provide the cell's name in the cell parameter and set isMulti to false. If you are reading multiple cells provide an Excel range, like A1:B5 as you would in Excel. The API will return JSON. Set isMulti to true. Examples:

Write requests

To write to a single cell, put the cell in the cell parameter and the value in the data parameter. Set isMulti to false. It will return a JSON response indicating success.

To write to multiple cells, set the cell parameter to the top left cell you are writing to. Set isMulti to true. Set data to a JSON array of the cells, where the top left in the JSON is the top left cell that you put.
