# API Overview

ExcelWebDatabases uses an API with GET requests which returns JSON. The API endpoint is:

The parameters are:

key is your API key. mode is read/write, defined by r or w. Document is the name of the document you would like to access. isMulti is a boolean, on the basis of whether you are accessing an individual cell or multiple cells in one request. cell is only provided if it is a cell request* and is an Excel cell, ex. A4, B1, or a range ex. A1:B2. data is only provided if it is a write request, and is JSON. column and row are only provided if it is a named request* and are names of the row and column that the cell which you are accessing is in.

*ExcelWebDatabases allows you to name rows and columns in your spreadsheet, or leave them unnamed. If your rows and columns are named, you may make requests using the row name and column name of the cell or the cell's name (ex. A1). If your rows and columns are unnamed, you must make the request using the cell's name.
