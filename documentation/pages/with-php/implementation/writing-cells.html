# Writing Cells

When writing to cells, you have the option to write to a single cell or to a range of cells.

# Writing using Cell Name

When writing to a single cell or multiple cells, you will receive JSON indicating success when successful. In the API request, set document to the name of the document you are accessing. To write to a single cell, set the parameters in the request as follows. Set key to your API key. Set data to the data you want to write to the cell. Set isMulti to false. Set mode to w. If your rows and colums are named, you can access the cell by providing row and column names (set row to the row name and column to the column name). # Writing Multiple Cells

To write to multiple cells, you must use the cell name to access the cell. Set the cell parameter to the top left cell you are writing to. Set isMulti to true. Set data to a JSON array of the cells, where the top left in the JSON is the top left cell that you put. And of course, set document to the document name, key to your API key, and mode to r.
