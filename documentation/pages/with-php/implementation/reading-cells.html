# Reading Cells

When making a read request, you have the option to read a single cell or to read multiple cells.

If you are reading a single cell, you will receive a JSON response with data set to the cell's value. In the API request, set document to the name of the document you are accessing. Set key to your API key. Set isMulti to false. Set mode to r. If your rows and colums are named, you can access the cell by providing row and column names (set row to the row name and column to the column name). # Reading Multiple Cells

If you are reading multiple cells, you must access the cells using a cell name range (ex. A1:B2). You cannot use rows and columns. In the API request, set document to the name of the document you are accessing. Set key to your API key. Set isMulti to true. Set mode to r. Finally, set cell to the cell range (ex.
