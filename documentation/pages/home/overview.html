ExcelWebDatabases Documentation

WHAT IS POSSIBLE?

With ExcelWebDatabases, you can upload an Excel spreadsheet and access it through a web API, making it like a web databases. It is theoretially usable via any language that is capable of making a web request, and a JS library is available.

This documentation will demonstrate the read/write capabilities of the ExcelWebDatabases API, such as...

  • Reading one cell at a time
  • Reading multiple cells at a time
  • Writing one cell at a time
  • Writing more than one cell at a time
