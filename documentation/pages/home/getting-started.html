How to use ExcelWebDatabases

Before setting up the programming, you first need to login to your ExcelWebDatabases account and upload the file you want to be your database. Keep the file name and the document name on hand.

Next, you need to connect your program to the API. The API uses GET requests, and will return JSON. A JS library is available, and PHP examples are also available. However, all that is necessary is a GET request. See "PHP examples" for the endpoint URL w/ parameters.
