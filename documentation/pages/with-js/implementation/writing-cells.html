# Writing In Cells

# Writing Singular Cells

Writing to cells uses the 7th parameter, the data parameter. To write to a single cell, you can either provide the cell name (eg. A1) or the row and column names. Set the data parameter with the data you want to put into the cell. If you are using row/column, set cell to 0. If you are using cell, set row and column to 0.
